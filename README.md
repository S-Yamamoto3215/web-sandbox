# web-sandbox

HTML、SCSS、TypeScript構成によるSandboxです。  

`yarn dev`でparcelが起動し、`http://localhost:1234`にindex.htmlが展開されます。  
index.htmlで読み込まれているSCSSやTypeScriptもコンパイルされた状態で読み込まれます。
